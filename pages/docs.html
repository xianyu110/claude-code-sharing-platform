Claude Code 合租服务使用文档

完整的 API 参考和使用指南

服务介绍

Claude Code 合租服务是一个专业的 AI 编程助手代理平台，通过合租模式为开发者提供经济实惠的 Claude API 访问服务。

高性能

99.9% 服务可用性，平均响应时间 < 500ms

安全可靠

企业级安全措施，API 密钥加密存储

经济实惠

通过合租模式降低 70% 的使用成本

简单易用

完全兼容 Claude API，无需修改现有代码

快速开始

1

注册账户

访问我们的 注册页面 创建账户并选择合适的套餐。

2

获取 API 密钥

登录到 控制台，在 API 密钥页面生成您的专属密钥。

3

开始使用

使用获得的 API 密钥，通过我们的代理端点访问 Claude 服务。

提示：我们的 API 完全兼容官方 Claude API，您只需要更改端点 URL 和 API 密钥即可。

身份认证

所有 API 请求都需要在请求头中包含您的 API 密钥：

请求头示例 
Authorization: Bearer your-api-key-here
Content-Type: application/json
安全提醒：请妥善保管您的 API 密钥，不要在客户端代码中直接暴露。

API 概览

基础 URL https://api.claude-sharing.com/v1
协议 HTTPS only
数据格式 JSON
字符编码 UTF-8

接口列表

聊天完成

POST /chat/completions

创建聊天完成请求，获取 Claude 的响应。

账户管理

GET /account/usage

获取当前账户的使用统计信息。

GET /account/limits

获取当前账户的配额和限制信息。

请求格式

聊天完成请求

POST /chat/completions 
{
  "model": "claude-3-sonnet-20240229",
  "messages": [
    {
      "role": "user",
      "content": "Hello, Claude! Can you help me write a Python function?"
    }
  ],
  "max_tokens": 1000,
  "temperature": 0.7,
  "stream": false
}

参数说明

参数 类型 必填 说明
model string 是 要使用的模型名称
messages array 是 对话消息列表
max_tokens integer 否 最大 token 数量，默认 1000
temperature float 否 响应随机性，0-1 之间，默认 0.7
stream boolean 否 是否使用流式响应，默认 false

JavaScript 示例

使用 fetch API 
async function callClaude(message) {
  const response = await fetch('https://api.claude-sharing.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer your-api-key',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'claude-3-sonnet-20240229',
      messages: [
        {
          role: 'user',
          content: message
        }
      ],
      max_tokens: 1000
    })
  });
  
  const data = await response.json();
  return data.content[0].text;
}

// 使用示例
callClaude('帮我写一个简单的计算器类')
  .then(result => console.log(result))
  .catch(error => console.error('错误:', error));

Python 示例

使用 requests 库 
import requests
import json

def call_claude(message):
    url = "https://api.claude-sharing.com/v1/chat/completions"
    headers = {
        "Authorization": "Bearer your-api-key",
        "Content-Type": "application/json"
    }
    
    payload = {
        "model": "claude-3-sonnet-20240229",
        "messages": [
            {
                "role": "user",
                "content": message
            }
        ],
        "max_tokens": 1000
    }
    
    response = requests.post(url, headers=headers, json=payload)
    
    if response.status_code == 200:
        return response.json()["content"][0]["text"]
    else:
        raise Exception(f"API 请求失败: {response.status_code} - {response.text}")

# 使用示例
try:
    result = call_claude("帮我写一个处理 CSV 文件的 Python 脚本")
    print(result)
except Exception as e:
    print(f"错误: {e}")

常见问题

如何获取 API 密钥？

注册账户后，登录到控制台，在 "API 密钥" 页面点击 "生成新密钥" 即可获取。每个账户最多可以生成 5 个 API 密钥。

支持哪些 Claude 模型？

我们支持所有最新的 Claude 3 模型，包括：

  • claude-3-opus-20240229（最强模型）
  • claude-3-sonnet-20240229（平衡模型）
  • claude-3-haiku-20240307（快速模型）

如何处理速率限制？

每个套餐都有相应的速率限制。如果遇到 429 错误，请稍后重试。建议在代码中实现指数退避重试机制。

数据是否安全？

我们采用企业级安全措施：所有数据传输使用 HTTPS 加密，API 密钥经过加密存储，不会记录或存储您的对话内容。
